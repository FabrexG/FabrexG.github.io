StyleApp — Recomendaciones de moda con IA

📅 2026 🏷 Android · IA · Fashion

Resumen

StyleApp es una aplicación móvil Android que genera recomendaciones personalizadas de atuendos integrando técnicas de Inteligencia Artificial con un sistema de gestión de guardarropa digital. El usuario registra las prendas que tiene y la app sugiere combinaciones según el clima, la ocasión y su historial de uso.

Arquitectura de la aplicación

La app sigue una arquitectura MVVM (Model-View-ViewModel) con un módulo de IA encapsulado como servicio local. El módulo de recomendación utiliza un modelo de embeddings de imágenes para representar cada prenda y un sistema de reglas combinado con filtrado colaborativo para generar sugerencias.

  • Frontend: Android nativo (Kotlin) con Jetpack Compose.
  • Módulo IA: TensorFlow Lite para inferencia on-device.
  • Base de datos: Room (SQLite) para el catálogo de prendas.
  • API de clima: integración con OpenWeatherMap para recomendaciones contextuales.

Módulo de recomendación

Cada prenda se representa mediante un vector de embedding extraído con MobileNetV2 fine-tuneado sobre un dataset de moda. La compatibilidad entre prendas se calcula mediante similitud del coseno en el espacio de embeddings, penalizada por reglas de estilo (color, textura, formalidad).

Estado del proyecto

El proyecto se encuentra en desarrollo activo. La versión alfa con funcionalidad básica de gestión de guardarropa y recomendación está disponible para pruebas internas. Se planea integrar feedback del usuario para mejorar las recomendaciones de forma personalizada.
